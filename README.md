# Airline Operations Control & Analytics Platform

##  Project Overview
An end-to-end data pipeline and analytics platform are developed for monitoring flights operation metrics. The solution combines information from multiple sources, analyzes the data with Snowflake, and displays insights on a Streamlit dashboard.

The platform is intended to help track flight delays, disruption trends, aircraft preparedness, etc.

---

##  Problem Statement
There is no centralization of airline operations information since they are stored in various resources, namely Oracle database, CSV files, and JSON events streams. As a result, it is hard to:

- Monitor flight delays in real time
- Identify patterns in disruptions
- Analyze aircraft preparedness
- Conduct centralized analytics

---

## Project Objective
- Develop a single data pipeline for airline operations
- Enable tracking of delays and disruptions
- Provide insights based on KPIs
- Ensure near real-time analytics capability
- Include AI-powered insights with Snowflake Cortex
